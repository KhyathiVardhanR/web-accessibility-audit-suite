Heading Best Practices

✅ One H1 Per Page

Use exactly one H1 for the main page title. Multiple H1s confuse screen readers.

✅ Don't Skip Levels

Go from H2 to H3, not H2 to H4. Skipped levels break the document outline.

✅ Descriptive Text

Headings should describe the section content. Avoid generic "Section 1" headings.

✅ Not Just for Styling

Use headings for structure, not size. Use CSS for visual styling instead.
